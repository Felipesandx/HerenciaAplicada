# HerenciaAplicada
Repositorio de practica de herencia, superclases y sobrescritura.

Este repositorio contiene una serie de ejercicios enfocados en la aplicación de Programación Orientada a Objetos (POO) en Java, específicamente en los conceptos de:

Herencia (extends)
Sobrescritura de métodos (@Override)
Encapsulamiento (private, protected)
Uso de super
Clases abstractas

Cada ejercicio representa un caso práctico diferente donde se modelan situaciones del mundo real mediante clases y relaciones de herencia.

## Características:
Uso de herencia para reutilizar atributos comunes.
Uso de sobrescritura en el método calcularTotal().
Validación para evitar valores negativos.
Aplicación de encapsulamiento en los atributos.
Uso de objetos dentro de clases (composición).

## Casos de Prueba

Se probaron distintos escenarios para validar el correcto funcionamiento:
Descuentos mayores al total (evitar valores negativos)
Recargos en facturas a crédito
Cálculo de cuotas
Sobregiro en cuentas corrientes
